Independent Samples

It's All About The Variance

Up to now, when comparing two groups, we considered a null hypothesis relying on the two groups coming from the same population.

Comparing Two Groups

Up to now, when comparing two groups, we considered a null hypothesis relying on the two groups coming from the same population. The question we were answering was:

If two samples are from the same population, what is the chance of finding at least this difference?

The chance here is quantified by a p-value

Null Hypothesis

The Null Hypothesis carries the assumption of equality, i.e.,
two groups come from the same underlying population $\Pi$.

We could write that: $\Pi(X) = \Pi_A(X) = \Pi_B(X)$

this implies that all the true statistics are assumed equal ($\bar{x}_A = \bar{x}_B$, $\mathrm{m}_A=\mathrm{m}_B$, etc..)

Formally, we write the Null hypothesis as Name: Hypothesis | Assumption, so:

$\mathrm{H}_0: \bar{x}_A = \bar{x}_B\, |\, \Pi_A(x) = \Pi_B(x)$

Recall that we stated xbar_1 - xbar_2 = delta0, where delta0 was the difference in the parameters (mu).

Comparing Two Groups

Student's T-Test

What we've learned so far is Student's T-Test, which tests the difference between the means of two groups, evaluating the null hypothesis that both samples were from the same underlying Gaussian distribution, N:
$\mathrm{H}_0: \bar{x}_A = \bar{x}_B | \mathcal{N}(\mu_A, \sigma_A) = \mathcal{N}(\mu_B, \sigma_B)$.
Assumption: A and B have the same "true variance". Thus it is called the equal variance t-test.
Similar to our $d$ statistic (difference), the T-Test use a t statistic, which has a "standardized" form (normalized to SE).
$t=\frac{\bar{x}_A-\bar{x}_B}{\sqrt{\frac{\sigma^2_\mathrm{AB}}{n_1} + \frac{\sigma^2_\mathrm{AB}}{n_2}}}$
Where $\sigma^2_{AB}$ is the "pooled" variance by combining two Normal distributions.

Comparing Two Groups

What if we only wanted to compare the centers of two samples, knowing full well that they come from different populations

By "different populations", we mean the variance, or other distributional parameter is unequal.

If two groups have unequal variances, it would not be reasonable to test if they came from the same underlying distribution.

Why?

They exists on two different scales!

The formulation for this Null Hypothesis is:

$$\mathrm{H}_0: \bar{x}_A = \bar{x}_B\, |\, p_A(x) \neq p_B(x)$$

Statistic of interest (e.g. $mean(X_1,X_2;-)$)

Sampling Distribution of the SOI

When $\mathrm{Var}(X_1)\neq\mathrm{Var}(X_2)$, we can use the unequal variance assumption. Known as Welch's t-test.

$$t = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{s^2_1}{n_1} + \frac{s^2_2}{n_2}}},$$ where $s^2$ is the sample variance.

With $\nu$ degrees of freedom: $$\nu = \frac{\left(\frac{s^2_1}{n_1} + \frac{s^2_2}{n_2}\right)^2}{\frac{(s^2_1/n_1)^2}{n_1-1} + \frac{(s^2_2/n_2)^2}{n_2-1}}$$

This is the Welch-Satterthwaite equation! Will be on the final!

Steps To Report Results Of $t$-test

  1. Calculate central tendency and 95% CI for each sample: Report $\overline{X}\,(95\%\, \mathrm{CI})$
  2. State the Null Hypothesis, and consider alternatives for tailness: $H_0:\Delta\overline{X}=0$, $H_a:$
  3. Calculate the observed difference: $\Delta\overline{X}$
  4. Calculate the Sum of the Weighted Variances: $\mathrm{SE}(\Delta\overline{X})=\sqrt{\frac{s^2_1}{n_1} + \frac{s^2_2}{n_2}}$
  5. Estimate the Degrees of Freedom
  6. Calculate the Effect Size 95% Confidence Interval
  7. Calculate the standardized distance between the samples: $t$
  8. Determine p-value and establish a critical region on the $t$-distribution
  9. Assess statistical significance

First fragment is "unequal variance" highlight.

Second frag is Welch's t, "because it's a t-test, we need a nu (degrees of freedom) parameter.

Note that the pooled variance is gone? We have a more complicated dof calculation that is a function of the sample variances.

Where Student's t-test relies on the underlying assumption that the two groups have the same variance, Welch's t-test relaxes this assumption. This is an attempt to resolve the Behrens-Fisher problem, but also suffers from approximation: the denominator "splits the difference" between the variances of the two groups.

Paired Samples

Paired samples (or dependent samples) occur when 2 or more data points are collected from the same observational unit.

Relationship Conditions For Paired Data

  • There is one pair of response values for each observational unit
  • There is a built-in comparison, each data point in one sample is linked to another data point in the other samples. E.g., Measuring machine performance every month.

Origin Of The Pairing

  • Matching similar individuals to create groups of two: paired design using matching.
  • Measuring the same individual (observational unit) in each condition: paired design using repeated measures.
When a paired design is possible, you typically get more informative results because units within a pair tend to be similar to each other, so differences within such each pair on the response variable tend to be due mainly to the explanatory variable.

Paired Samples

Types and Conditions

Pre-test/post-test samples in which a factor is measured before and after an intervention.

Cross-over trials in which individuals are randomized to two treatments and then the same individuals are crossed-over to the alternative treatment.

Matched samples, in which subjects are matched on inherent characteristics (age, mfr. lot#, etc.).

Duplicate measurements on the same subject.

Any circumstance in which each data point in one sample is uniquely matched to a data point in the second sample.

Paired-Sample Analysis

The key to analyzing paired data is to work with the difference within each pair of data values rather than between the two original samples.
This helps us to eliminate the variability across different units and instead focus on what we really care about: the difference between the maximum frictional forces across formulas.

Old Formula 5.30 5.64 5.60 6.22 5.14 5.24 5.71 5.78 6.07 5.57 5.76 5.99 5.668 0.317
New Formula 5.13 5.33 5.68 5.50 5.32 5.17 5.28 5.75 5.96 5.23 5.55 5.69 5.466 0.252
Difference -0.17 -0.31 0.08 -0.72 0.18 -0.07 -0.43 -0.03 -0.11 -0.34 -0.21 -0.30 -0.202 0.231

Note: the mean of the sample of differences is equal to the difference of the sample means: $\bar{x}_\textrm{Difference} = \bar{x}_\textrm{new}-\bar{x}_\textrm{old}$

We calculate the t-statistic as $\frac{\bar{d}}{SE_{d}} = \frac{\overline{\Delta x}}{s_{d}/\sqrt{N}}$.

Lubricant Selection

Null Hypothesis: The mean difference, $\mu_D$, is $0\therefore$ $$H_0:\mu_D=0.$$

Alternative Hypothesis: $$H_a: \mu_D\neq0.$$Two-tailed test!

Calculate t-statistics for Difference: $t_D=\frac{\bar{d}}{s_D/\sqrt{n}}=\frac{-0.202}{0.231/\sqrt{12}}=-3.031$

Degrees of freedom: $\nu = n-1 = 12-1 = 11$

Check p-value against the t-distribution for $\alpha=0.05$: $p=0.0114$ Reject $H_0$!

Conclusions:

With a p-value less than 0.05, and based on the design of the study, we have strong evidence against the null hypothesis and conclude that the new formula significantly alters maximum rotational friction in these machines.

What about the outliers? Check those machines for other factors.

Generally, we can draw cause-effect conclusion since the researchers used random assignment for the order and the selection of the machines. If they hadn't, what might be a confounding factor? Heat, fatigue between trials...

Can we generalize to the rest of the machines at the company? Probably safe.
